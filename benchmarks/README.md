# Benchmarks

This directory contains early synthetic benchmark scripts and example outputs for `mechanopharm-infer` `v0.0.1`.

Included:
- `generate_synthetic.py`: create clean and noisy synthetic datasets based on the minimal reference models
- `run_clean_benchmark.py`: run the current inference workflow on clean synthetic cases
- `outputs/`: example generated datasets and clean-benchmark outputs

Note:
The current benchmark outputs are intended as development-facing examples for the public `v0.0.1` release. They help illustrate current behavior and known limitations of the rule-based workflow, including places where evidence-flag definitions may still be refined.
